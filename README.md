# MCOO

一个基于 VitePress 构建的多功能文档平台，包含知识库、博物馆、项目库等多个栏目，提供丰富的自定义组件和工具。

## 📋 项目简介

MCOO 是一个综合性文档平台，旨在提供各种技术文档、工具集和项目展示。通过 VitePress 构建，具有现代化的界面和良好的用户体验。

## 📁 项目结构

```
MCOO/
├── .github/           # GitHub Actions 工作流配置
├── .vitepress/        # VitePress 配置和主题
│   ├── configs/       # 配置文件
│   ├── theme/         # 自定义主题
│   └── config.mjs     # 主配置文件
├── docs/              # 文档内容
│   ├── 00.杂七杂八/   
│   ├── 01.知识库/     
│   ├── 98.博物馆/     
│   └── 99.项目库/     
├── public/            # 静态资源
├── tools/             # 工具集
└── README.md          # 项目说明
```

## 🎯 主要功能

- **知识库**：包含 VitePress、Markdown、CloudFlare 等技术文档
- **自定义组件**：BiliVideo、ColorConverter、Linkcard 等丰富组件
- **在线工具**：MD 编辑器、表格编辑器、颜色转换器等
- **现代化主题**：自定义主题样式，支持多种视觉效果
- **响应式设计**：适配各种设备屏幕

## 🚀 快速开始

### 安装依赖

```bash
npm install
```

### 本地开发

```bash
npm run docs:dev
```

### 构建生产版本

```bash
npm run docs:build
```

## 📦 技术栈

- **VitePress**：静态站点生成器
- **Vue 3**：前端框架
- **TypeScript**：类型系统
- **GitHub Actions**：自动化部署
